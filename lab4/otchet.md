# Лабораторная работа №4  
## Выделение контуров на изображении
### Вариант 5 - Оператор Шарра 3х3

---

### Оператор Шарра

Для вычисления градиентов используются маски:

Gx:
```
3 10 3 
0 0 0
-3 -10 -3
```

Gy:
```
3 0 -3
10 0 -10
3 0 -3
```

---

### Вычисление градиента

Градиент вычисляется по формуле:

\[
G = \sqrt{G_x^2 + G_y^2}
\]

---

### Нормализация

Полученные значения приводятся к диапазону [0, 255] для корректного отображения.

---

### Бинаризация

Пороговая обработка выполняется по правилу:

- если \( G > T \) → 255  
- иначе → 0  

В работе использовались значения порога:

**T = 20, 30, 40, 50, 60, 70**

## Анализ результатов

В ходе эксперимента было выявлено:

- При малых значениях порога (20–30) наблюдается большое количество шумов  
- При средних значениях (40–50) достигается оптимальное выделение контуров  
- При высоких значениях (60–70) остаются только наиболее выраженные границы  

Таким образом, выбор порога существенно влияет на качество результата.
